# Календарь соревнований по фехтованию

Приложение для просмотра и фильтрации соревнований по фехтованию с бесконечной подгрузкой данных.

## 🚀 Быстрый старт

### Установка и запуск

# Клонирование репозитория
git clone https://github.com/o1901k/pointTest.git
cd calendar

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start


### Переменные окружения

Создайте файл `.env` в корне проекта:

REACT_APP_API_URL=https://api-url.com
REACT_APP_API_USERNAME=your_username
REACT_APP_API_PASSWORD=your_password
```

## 🏗 Архитектура

Приложение построено на **React** с использованием **TypeScript** и следует модульной архитектуре. Основные компоненты:

- **`/components`** — UI-компоненты: фильтры, таблица соревнований, состояния загрузки/ошибок
- **`/hooks`** — кастомные хуки: `useInfiniteScrollPaginated` для бесконечной подгрузки с пагинацией
- **`/services`** — API-слой с кэшированием регионов и трансформацией данных
- **`/types`** — хранение TypeScript-типов для данных API
